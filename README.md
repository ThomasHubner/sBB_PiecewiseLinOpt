# sBB_PiecewiseLinOpt
Spatial branch-and-bound algorithm for piecewise-linear optimization problems.

This repository is an accompaniment to the paper titled *Spatial branch-and-bound for nonconvex piecewise-linear optimization*.
It contains the code used to conduct computational experiments. 
Details and results can be found in Section 4 and the Appendix A3 of the above mentioned paper.
